Find My Peep is a crowd-aware navigation system designed to help attendees reconnect with friends inside large and crowded venues like stadiums.

The application simulates real-time user locations and intelligently suggests an optimal meeting point that minimizes walking distance while avoiding congested areas. It uses a visual grid-based map with intuitive emojis to represent users, crowded zones, and recommended meeting spots.
